## 系统概述

### 简介
项目进度管理系统是一个功能完整的团队协作平台，支持项目管理、任务跟踪、甘特图可视化、实时协作等工作流功能。

### 技术栈
- **前端**: Vue 3 + Element Plus + TypeScript + Pinia + Vite
- **后端**: FastAPI + SQLAlchemy + JWT + WebSocket
- **数据库**: MySQL / SQLite / SQL Server
- **实时通信**: WebSocket
- **国际化**: i18next

### 系统要求
- Node.js 18+
- Python 3.9+
- MySQL 8.0+ (可选)

## 快速开始

### 1. 安装依赖

#### 后端
```bash
cd backend
pip install -r requirements/base.txt
```

#### 前端
```bash
cd frontend
npm install
```

### 2. 配置环境

复制环境配置文件：
```bash
cp .env.example .env
```

编辑 `.env` 文件，配置数据库连接和邮件服务。

### 3. 启动服务

#### 开发环境
```bash
# 后端
cd backend
uvicorn app.main:app --reload --port 8000

# 前端
cd frontend
npm run dev
```

#### Docker 部署
```bash
docker-compose up -d
```

## 操作指南

### 1. 登录系统
1. 打开登录页面
2. 输入用户名和密码
3. 点击登录按钮

### 2. 创建项目
1. 进入项目管理页面
2. 点击"新建项目"按钮
3. 填写项目名称、描述、时间计划
4. 点击"创建"按钮

### 3. 管理任务
1. 进入项目详情页面
2. 点击"添加任务"按钮
3. 填写任务信息（标题、描述、负责人、截止日期）
4. 拖拽任务到不同状态列

### 4. 使用甘特图
1. 进入甘特图视图
2. 查看项目时间线
3. 拖拽任务调整时间
4. 点击任务查看详情

### 5. 使用看板视图
1. 进入看板视图
2. 拖拽任务卡片到不同列
3. 点击列标题查看统计

### 6. 发起审批流程
1. 打开工作流模块
2. 点击"新建审批"
3. 选择审批类型
4. 提交审批请求

### 7. 管理团队成员
1. 进入团队协作模块
2. 点击"添加成员"
3. 设置成员角色
4. 分配项目权限

### 8. 数据备份
1. 进入系统设置
2. 点击"数据备份"
3. 选择备份类型
4. 执行备份操作

### 9. 切换语言
1. 点击右上角设置
2. 选择"语言设置"
3. 选择目标语言
4. 页面自动刷新

### 10. 切换主题
1. 点击右上角主题图标
2. 选择深色/浅色模式
3. 主题即时生效

## 常见问题

### Q1: 如何重置密码？
A: 联系系统管理员重置密码。

### Q2: 如何邀请新成员？
A: 在项目设置中添加成员，输入其邮箱地址。

### Q3: 任务无法分配？
A: 确保被分配的成员已经是项目成员。

### Q4: 导出文件过大？
A: 建议分批导出或使用筛选条件减少数据量。

### Q5: 邮件通知收不到？
A: 检查垃圾邮件文件夹，确认邮件服务配置正确。

### Q6: 如何创建审批流程？
A: 进入工作流模块，点击新建审批，选择模板后提交。

### Q7: 看板视图如何调整任务状态？
A: 拖拽任务卡片到目标列即可自动更新状态。

### Q8: 数据如何备份？
A: 进入系统设置→数据备份，选择手动或自动备份。
