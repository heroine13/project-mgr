## 1. 系统概述

### 1.1 系统简介
项目进度管理系统是一个功能完整的团队协作平台，支持项目管理、任务跟踪、甘特图可视化、实时协作等工作流功能。

### 1.2 技术栈
| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Element Plus + TypeScript + Pinia + Vite |
| 后端 | FastAPI + SQLAlchemy + JWT + WebSocket |
| 数据库 | MySQL / SQLite / SQL Server |
| 实时通信 | WebSocket |
| 国际化 | vue-i18n |

### 1.3 系统环境
- **Node.js**: 18+
- **Python**: 3.9+
- **MySQL**: 8.0+ (生产环境)

## 2. 快速开始

### 2.1 测试环境访问
| 服务 | 地址 |
|------|------|
| 前端 | http://test.prjmanger.goldfon.cn 或 http://localhost:5173 |
| 后端API | http://test-api.prjmanger.goldfon.cn 或 http://localhost:8000 |
| API文档 | http://localhost:8000/docs |

### 2.2 Docker启动（本地开发）
```bash
# 启动后端
cd backend
docker build -t project-mgr-backend -f Dockerfile.dev .
docker run -d -p 8000:8000 project-mgr-backend

# 启动前端
cd frontend
docker build -t project-mgr-frontend -f Dockerfile.dev .
docker run -d -p 5173:3000 project-mgr-frontend
```

### 2.3 本地开发启动
```bash
# 后端
cd backend
pip install -r requirements/base.txt
uvicorn app.main:app --reload --port 8000

# 前端
cd frontend
npm install
npm run dev
```

## 4. 操作指南

### 4.1 登录系统
1. 打开登录页面 http://localhost:5173/login
2. 输入用户名和密码
3. 点击登录按钮
4. 成功后将跳转到仪表盘

### 4.2 创建项目
1. 登录后进入仪表盘，点击"新建项目"
2. 或通过菜单：项目管理 → 创建项目
3. 填写项目名称、描述、时间计划
4. 点击"创建"按钮

### 4.3 管理任务
1. 进入项目详情页
2. 点击"添加任务"
3. 填写任务信息：标题、描述、负责人、截止日期、优先级
4. 点击"保存"

### 4.4 使用甘特图
1. 进入项目详情
2. 点击"甘特图"标签
3. 查看项目时间线
4. 拖拽任务调整时间

### 4.5 使用看板视图
1. 点击顶部导航"看板"
2. 拖拽任务卡片到不同列
3. 状态自动更新

### 4.6 管理用户和权限
1. 进入"用户管理"页面
2. 用户Tab：增删改查用户、分配角色
3. 角色Tab：创建/编辑角色
4. 权限Tab：配置模块权限（树形复选框）
5. 部门Tab：维护组织架构

### 4.7 数据导出
1. 进入相应模块（项目/任务/报表）
2. 点击"导出"按钮
3. 选择格式（Excel/CSV/PDF）
4. 下载文件

### 4.8 切换语言
1. 登录页或系统右上角
2. 点击语言选择器
3. 选择"中文"或"English"
4. 页面自动刷新

### 4.9 切换主题
1. 点击右上角主题图标
2. 选择深色/浅色模式
3. 主题即时生效

## 6. 常见问题

### Q1: 登录失败，提示"Invalid username or password"
**A**: 
- 确认账号密码正确（参考测试账号表）
- 确认账号已激活（is_active=true）
- 检查后端服务是否正常运行

### Q2: 登录后一直停留在登录页
**A**: 
- 清除浏览器缓存
- 按 Ctrl+F5 强制刷新
- 检查浏览器控制台是否有错误

### Q3: 页面显示白屏
**A**: 
- 检查浏览器控制台错误
- 确认前端服务正常运行
- 尝试清除缓存后刷新

### Q4: API请求返回404
**A**: 
- 检查后端服务是否启动
- 确认API路径正确
- 查看后端日志

### Q5: 如何重置密码？
**A**: 联系系统管理员，或使用忘记密码功能

### Q6: 如何邀请新成员？
**A**: 在用户管理中添加新用户，分配角色和部门

### Q7: 任务无法分配给某些成员
**A**: 确认该成员已是项目成员，在项目设置中添加

### Q8: 数据导出失败
**A**: 
- 检查导出数据量是否过大
- 确认后端已安装导出依赖（openpyxl等）

### Q9: 邮件通知收不到
**A**: 
- 检查垃圾邮件文件夹
- 确认邮件服务配置正确
- 查看后端邮件发送日志

### Q10: 看板视图无法拖拽任务
**A**: 
- 确认浏览器支持拖拽
- 检查浏览器控制台是否有JS错误
