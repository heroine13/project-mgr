## 系统概述

### 简介
项目进度管理系统是一个功能完整的团队协作平台，支持项目管理、任务跟踪、甘特图可视化、实时协作等功能。

### 技术栈
- **前端**: Vue 3 + Element Plus + TypeScript
- **后端**: FastAPI + SQLAlchemy
- **数据库**: MySQL / SQLite / SQL Server
- **实时通信**: WebSocket

### 系统要求
- Node.js 18+
- Python 3.9+
- MySQL 8.0+ (可选)

## 快速开始

### 1. 安装依赖

#### 后端
```bash
cd backend
pip install -r requirements.txt
```

#### 前端
```bash
cd frontend
npm install
```

### 2. 配置环境

复制环境配置文件：
```bash
cp .env.example .env
```

编辑 `.env` 文件，配置数据库连接和邮件服务。

### 3. 启动服务

#### 开发环境
```bash
# 后端
cd backend
uvicorn app.main:app --reload

# 前端
cd frontend
npm run dev
```

访问地址：
- 前端: http://localhost:5173
- 后端 API: http://localhost:8000
- API 文档: http://localhost:8000/docs

#### Docker 部署
```bash
docker-compose up -d
```

## 操作指南

### 1. 登录系统
1. 打开登录页面
2. 输入用户名和密码
3. 点击登录按钮

### 2. 创建项目
1. 进入项目管理页面
2. 点击"新建项目"按钮
3. 填写项目名称、描述、时间计划
4. 点击"创建"按钮

### 3. 管理任务
1. 进入项目详情页面
2. 点击"添加任务"按钮
3. 填写任务信息（标题、描述、负责人、截止日期）
4. 拖拽任务到不同状态列

### 4. 使用甘特图
1. 进入甘特图视图
2. 查看项目时间线
3. 拖拽任务调整时间
4. 点击任务查看详情

### 5. 协作评论
1. 打开任务详情页面
2. 在评论区输入评论
3. 使用 @提及成员
4. 使用表情反应

### 6. 导出数据
1. 进入数据页面
2. 点击"导出"按钮
3. 选择导出格式 (Excel/CSV/PDF)
4. 下载导出文件

### 7. 设置通知偏好
1. 点击右上角通知图标
2. 选择"通知设置"
3. 开启/关闭各类通知
4. 设置邮件接收频率

## 常见问题

### Q1: 如何重置密码？
A: 联系系统管理员重置密码。

### Q2: 如何邀请新成员？
A: 在项目设置中添加成员，输入其邮箱地址。

### Q3: 任务无法分配？
A: 确保被分配的成员已经是项目成员。

### Q4: 导出文件过大？
A: 建议分批导出或使用筛选条件减少数据量。

### Q5: 邮件通知收不到？
A: 检查垃圾邮件文件夹，确认邮件服务配置正确。
