# 项目进度管理系统 - 系统分析文档

## 一、系统目标与范围

### 1.1 系统目标
为中小型工厂提供一套完整的项目进度管理解决方案，实现：
- 项目全生命周期管理
- 任务分配与进度追踪
- 团队协作与资源调度
- 数据可视化与报表分析

### 1.2 系统范围
- **用户群体**：项目管理人员、团队成员、客户
- **功能模块**：30+功能模块，涵盖项目、任务、团队、文档、报表等
- **部署方式**：支持Docker容器化部署

## 四、业务流程

### 4.1 项目创建流程
```
1. 项目经理创建项目
   ↓
2. 填写项目基本信息（名称、描述、时间、预算）
   ↓
3. 添加团队成员
   ↓
4. 创建任务清单
   ↓
5. 设置任务依赖关系
   ↓
6. 任务指派给团队成员
   ↓
7. 项目启动
```

### 4.2 任务执行流程
```
1. 团队成员收到任务通知
   ↓
2. 查看任务详情
   ↓
3. 开始执行任务
   ↓
4. 如有问题，可在任务下评论讨论
   ↓
5. 完成任务，变更状态为"已完成"
   ↓
6. 项目经理审核
   ↓
7. 任务关闭
```

### 4.3 审批流程（工作流）
```
1. 提交审批申请
   ↓
2. 审批人收到通知
   ↓
3. 审批人审核（通过/拒绝/退回修改）
   ↓
4. 如被拒绝，返回申请人修改
   ↓
5. 审批通过，流程结束
   ↓
6. 通知申请人结果
```

## 七、数据字典

### 7.1 用户表 (users)
| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | Integer | 主键 | PK, AUTO_INCREMENT |
| username | String(50) | 用户名 | NOT NULL, UNIQUE |
| email | String(100) | 邮箱 | NOT NULL, UNIQUE |
| hashed_password | String(255) | 加密密码 | NOT NULL |
| full_name | String(100) | 姓名 | NULL |
| is_active | Boolean | 是否激活 | DEFAULT TRUE |
| is_superuser | Boolean | 是否超管 | DEFAULT FALSE |
| role_id | Integer | 角色ID | FK -> roles.id |
| created_at | DateTime | 创建时间 | DEFAULT NOW() |
| updated_at | DateTime | 更新时间 | 自动更新 |

### 7.2 项目表 (projects)
| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | Integer | 主键 | PK |
| name | String(200) | 项目名称 | NOT NULL |
| description | Text | 项目描述 | NULL |
| code | String(50) | 项目编号 | UNIQUE |
| status | String(50) | 项目状态 | DEFAULT 'planning' |
| owner_id | Integer | 负责人 | FK -> users.id |
| start_date | DateTime | 开始日期 | NULL |
| end_date | DateTime | 结束日期 | NULL |
| budget | Integer | 预算 | NULL |
| created_at | DateTime | 创建时间 | DEFAULT NOW() |

### 7.3 任务表 (tasks)
| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | Integer | 主键 | PK |
| title | String(200) | 任务标题 | NOT NULL |
| description | Text | 任务描述 | NULL |
| status | String(11) | 任务状态 | DEFAULT 'pending' |
| priority | String(6) | 优先级 | DEFAULT 'medium' |
| assignee_id | Integer | 指派人 | FK -> users.id |
| project_id | Integer | 所属项目 | FK -> projects.id |
| due_date | DateTime | 截止日期 | NULL |
| estimated_hours | Integer | 预计工时 | NULL |
| actual_hours | Integer | 实际工时 | NULL |
| created_by | Integer | 创建人 | FK -> users.id |
